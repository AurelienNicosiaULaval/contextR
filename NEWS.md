# contextR 0.2.0

- Introduced S3 generic `context()` with class-specific methods.
- Added unified `contextual` output object with traceability fields.
- Added backend abstraction with default mock backend (`context_backend_*`).
- Added strict/safe/free safety checks and `context_audit()`.
- Added compatibility wrappers with lifecycle deprecation warnings.
- Added tests, vignettes, and GitHub Actions R-CMD-check workflow.
